Downloads

This is the latest version of the combined downloader/firmware upgrader app.

Windows

Windows is fully supported and this version has had the most testing. The installer contains everything you need. Run it, and then you'll find openaltimeter in your start menu.

openaltimeter_win_2.1.0.0.exe - 02 January 2012

Mac OS X

This Mac OS version supports all features that the Windows version does, but has not had as much testing. You should be able to use the Mac OS version to do everything, but you may need to fall back on the Windows version in case of bugs. The Mac version is known to work on OS X 10.7.

You might find that you need to install the FTDI drivers for the serial dongle manually before it will show up in the OA serial port menu. If so, you can download the drivers from here.

openaltimeter_macos_2.1.1.0.zip - 14 March 2015

Firmware

If you're using the firmware upgrade function of the downloader then you no longer need to download the firmware separately. It's here nonetheless for your convenience!

openaltimeter_firmware_V8.zip - 02 January 2012
